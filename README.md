# 🗞 RSS Smart Scraper

## 🎯 Objectif
Ce script Python permet de scanner des centaines de flux RSS à la recherche d’actualités contenant des mots-clés donnés. Il affiche les articles pertinents dans la console et les enregistre dans un fichier `resultat.txt`.

---

## 🚀 Fonctionnalités
- Chargement des flux RSS depuis un fichier `rss_list.txt`
- Lecture des mots-clés depuis `mots_cles.txt`
- Récupération des titres, dates et liens des articles
- Filtrage selon les mots-clés
- Exécution asynchrone pour de bonnes performances
- Résultats affichés dans la console et enregistrés dans un fichier texte
